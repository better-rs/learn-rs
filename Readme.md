## 准备工作:

- 配置 rust 开发环境
- 安装 Clion

### Clion:

- https://www.jetbrains.com/help/clion/rust-support.html#tests
- https://plugins.jetbrains.com/plugin/8182-rust/docs/rust-testing.html
- 基于 unit test 方式编写代码.
- Clion IDE 支持直接运行 test 代码, 方便快速测试语法点
